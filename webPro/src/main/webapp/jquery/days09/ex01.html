jQuery는 ㄴ 빠르고 작으며 풍부한 기능을 갖춘 JavaScript 라이브러리 ㄴ 다양한 브라우저에서 작동하는 사용하기 쉬운 API ㄴ HTML 문서 탐색 및 조작, 이벤트 처리, 애니메이션, Ajax 등을 훨씬 간편하게 구현 1) 다운로드 2) CDN( Content Delivery Network) 분산된 서버에서 jQuery를 호스팅하여 성능상의 이점을 제공하는 방식. [jquery 구문 형식] $(jquery selector).action(); 1) jquery selector 정리 2) action() 정리 [***] 웹 페이지에서 jquery 외에 다른 프레임워크를 사용할 때 jquery에서 사용하는 $ 기호를 다른 프레임워크에서 다른 의미의 변수로 벌써 사용하고 있다면, $ 기호의 충돌이 발생할 것이다. 다른 프레임워크( Angular, knokout, ember, backbone 등등) $를 다른 의미로 사용할 수 있다.
